# Rename from tfimportgen to tfimport-cli

## Summary

The tool has been successfully renamed from **tfimportgen** to **tfimport-cli**.

## Changes Made

### 1. Go Module
- Updated `go.mod`: `module github.com/njingu90/tfimport-cli`
- Updated all import statements in Go files to use new module path

### 2. Directory Structure
- Renamed: `/workspaces/tfimportgen` → `/workspaces/tfimport-cli`
- Renamed: `cmd/tfimportgen` → `cmd/tfimport-cli`

### 3. Binary Name
- New binary name: `tfimport-cli`
- Build output: `./bin/tfimport-cli`

### 4. Code References
Updated in the following files:
- `Makefile` - Binary name, help text, build targets
- `internal/cmd/version.go` - Version output
- `internal/cmd/flags.go` - Usage/help text and examples
- `pkg/importgen.go` - Generated file header comment
- All `*.go` files - Import paths
- All `*.md` files - Documentation references

### 5. Tests
- Updated test expectations for new separators (already fixed)
- Fixed `TestGetAllResources` to use managed resource instead of data source
- All tests passing ✅

## Usage

### Old
```bash
tfimportgen --version
tfimportgen analyze --state terraform.tfstate
tfimportgen generate --state terraform.tfstate
```

### New
```bash
tfimport-cli --version
tfimport-cli analyze --state terraform.tfstate
tfimport-cli generate --state terraform.tfstate
```

## Build & Test

```bash
# Build
make build

# Run tests
make test

# Install
make install

# Multi-platform build
make build-all
```
